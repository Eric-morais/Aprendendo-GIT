Olá esse arquivo contém informaçoes sobre os meus primeiros passos no GIT.
-----------------------------------------------------------------------------Oque é GIT ? R:o Git é um sistema de controle de versão distribuído que permite acompanhar o histórico de alterações de arquivos e projetos. Ele é usado principalmente no desenvolvimento de software, mas pode ser usado para qualquer tipo de arquivo.
-----------------------------------------------------------------------------

1- Instalação - https://git-scm.com/downloads
2- Após a instalação verifiquei se esta tudo ok com o comando git -help ou git --version. entao após a instalação abrir prompt de comando (CMD) e digitei o comando acima.
3- Criei uma pasta (nomeei com nome qualquer), pois ela será o repositório que irei utilizar. após criada e dentro da pasta cliquei com botao direito e cliquei no Open Git bash here, nesse momento  abriu o terminal git, Digitei  git init a príncipio não notei nenhuma mudança, mas nesse momento foi criado um arquivo .git ele esta oculto, para enxergar foi necessário desocultar na visualizacao da pasta.
4- Para que eu possa verificar o status do meu repositorio vou digitar o comonado git status, nesse comando mostrou todas as possíveis alteraçoes e inclussao que podem ter ocorrido.
5- Para Adicionar o arquivo no controle de versão basta digitar o comando git add . ou colocar o nome do arquivo tem que estar dentro de "nome do arquivo", para ser mais rapido é melhor colocar git add . pois sera salvo todos os arquivos adicionados ou alterados
6- Após realizar o procedimento acima devemos realizar o procedimento de commit esse procedimento permite adicionarmos em poucas palavras o que fizemos como se fosse um comentario, o comando é git commit -m "comentario"
7- Após o commit, para enviar os arquivos para nuvem é necessario realizar um outro comando chamando git push, porem para realizar esse procedimento é necessario ter realizado a devida configuração para os arquivos irem para repositorio na nuvem (GITHUB) e esse procedimento é git remote add origin link pego la no repositorio do github.
8- Para ver o historico das versoes é necessario digitar o comando git reflog esse comando permite verificar todas as versoes que foi versionada. para alterar as versao voltando para anterior a um comando que é:
9- Git reset --hard (codigo obtido pelo comando git reflog) apertar enter e a alteracao para a versao anterior sera feita para voltar ao que era basta fazer o mesmo prossedimento com novo codigo que o reflog mostrara
10- para verificar BRANCH (pasta) em que está é necessario digitar o comando git branch e apos ele listara as brench disponivel notamos que a branch em que estiver selecinada ela estará com asteristico* ao lado.
11- para criar uma branch nova utilizara o comando git Branche e o nome da branch  
12- para alterar a branch é necessario um comando git checkout e o nome da branch que quero ir , apartir daí tudo que for alterado ou acresentado ela sera salvo na branch nova.
